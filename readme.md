# AEG Fire Counter 

## Prerequirement

### Hardware

* MCU: PIC16F1508/PIC16F1509
* Display: OLED Display based on SSD1306 128\*64 or 128\*32
* Senser: Photointerrupter / Photoreflector

### Build environment

* MPLAB X v5.05
* XC8 Compiler v2.00
* MPLAB Code Configurator (MCC) v3.65.1
* Pickit3 or compatible debugger/programmer
